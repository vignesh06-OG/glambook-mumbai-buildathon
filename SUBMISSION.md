## 1. Project title

**Mumbai Salon Marketplace** — AI-powered beauty salon discovery & booking

---

## 2. Short description (2–3 lines)

A city-focused startup-style platform where Mumbai users browse salons, filter by area/service/home visits, get AI-matched recommendations from plain English queries, and book appointments in under a minute.

---

## 3. AI tools & workflow used

| Phase | Tool | What we did |
|-------|------|-------------|
| Scaffold | `create-next-app` | Next.js 16 + TypeScript + Tailwind |
| UI & logic | **Cursor** + Claude | Components, pages, booking flow, filters |
| Content | **ChatGPT** | Salon names, areas, service copy (seed data) |
| AI feature | Custom matcher | Keyword + intent scoring in `src/lib/ai-recommend.ts` |
| Design | Tailwind | Mobile-first rose/fuchsia salon brand |
| Deploy | **Vercel** | One-click from GitHub |

**Innovation:** Smart Salon Finder accepts queries like *"bridal makeup near Juhu"* or *"home facial Thane"* and ranks salons without a paid API — demo-ready for hackathon judges.
